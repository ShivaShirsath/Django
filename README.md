# Django
### Installing 
   ```bash
   pip install django
   ```
   Or
   ```bash
   pip install django --upgrade
   ```
### Create Project
   ```bash
   django-admin startproject PROJECT_NAME
   ```
### Open Project
   ```bash
   cd PROJECT_NAME
   ```
### Run Server
   ```bash
   python manage.py runserver
   ```
   [http://127.0.0.1:8000](http://127.0.0.1:8000)
