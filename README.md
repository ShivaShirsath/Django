# Django
### Install
   ```bash
   pip install django
   ```
### Upgrade
   ```bash
   pip install django --upgrade
   ```
### Create Project
   ```bash
   django-admin startproject PROJECT_NAME
   ```
### Open Project
   ```bash
   cd PROJECT_NAME
   ```
### Run Server
   ```bash
   python manage.py runserver
   ```
### Open Server : http://127.0.0.1:8000
